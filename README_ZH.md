一个自动从网络上收集tracker服务器的工具。

### 特性

- 自动化收集 tracker 信息
- 定期更新 tracker 列表
- 智能去重

### 使用方法
1. 确保你已安装 Python 3.10+
2. 复制本项目到本地
3. 编写配置文件(config.ini)  
   [具体介绍](/docs/config_ZH.md)
4. 运行 `main.py`
